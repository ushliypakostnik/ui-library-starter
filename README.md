# Vue Components UI Library Starter

### Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Development documentation
```
npm run docs:dev
```

### Build documentation
```
npm run docs:build
```
