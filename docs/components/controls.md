# controls.js

## Description

Mixin additional functionality for active controls.

## API

### Props

| **Name** | **Type**     | **Description**       | **Default** |
| :------- | :----------- | :-------------------- | ----------: |
| **value** | [String, Number] | Sync value | **required** |
| **onClick** | Function | Callback on click | `() => {}` |

### Events

| **Name** | **Description** |
| :------- | :-------------- |
| **change** | Emit on value change |
| **click** | Emit on click |
